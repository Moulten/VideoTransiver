Reciver - Клиент, принимающий данные и воспроизводящий принятое видео.

Sender - Однопоточный сервер, отправляющий видеоданные клиенту.

server - Многопоточный сервер, отправляющий видеоданные клиенту.

Исполняемые файлы сохраняются в директории bin

Сервер запускается командой ./Server sam.mov(или любое другое 
имя видеофайла, лежащего в той же директории)

Клиент запускается командой ./Reciver 127.0.0.1

Для закрытия окна с видео до его окончания используется кнопка Esc
